TastyPlug - The plug.dj extension.

Use the UserScript for the quickest updates and the best experience.

All of them, except for the bookmark, start automatically when you enter a room on plug.dj. If you wish to start it manually instead, drag the "TastyPlug" button to your bookmarks or right click it to save as a bookmark.

Hover over a button to see more information.

Features

  • AutoWoot
  • AutoJoin
  • Hide Video
  • Legacy Chat
  • Booth Alert
  • Chat Images
  • Custom Emotes
  • Chat Mentions
  • Join Notifications
